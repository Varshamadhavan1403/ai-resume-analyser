# AI Resume Analyzer

An AI-powered platform that analyzes resumes against job descriptions and provides:

- Resume-job match score
- Skill gap analysis
- AI recommendations
- Interview question generation

## Tech Stack

- Python
- Django
- Django REST Framework
- PostgreSQL
- Celery
- Redis
- OpenAI / Gemini API

## Features

### Phase 1
- User Authentication
- Resume Upload
- Resume Parsing

### Phase 2
- Job Description Analysis
- Match Score Engine
- Skill Gap Detection

### Phase 3
- AI Recommendations
- Interview Question Generator
- Dashboard Analytics

## Setup

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
